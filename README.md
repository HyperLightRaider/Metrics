# Metrics

MVP веб-приложения «Metrics», реализующий полный цикл работы с данными пользователя. Регистрация, ежедневный ввод ключевых показателей, их визуализация в виде графиков и упрощенный модуль аналитики, демонстрирующий наиболее сильную корреляцию между метриками пользователя.

## Цель проекта

Разработать прототип веб-приложения, который позволяет всем, кто следит за своим состоянием, решать проблему разрозненности данных и отсутствия аналитики. Приложение помогает за минуту в день фиксировать ключевые показатели (сон, энергия, настроение, продуктивность), видеть динамику на графиках и автоматически получать статистику о том, какие факторы реально влияют на самочувствие.



## Технологии

\- Frontend: HTML, CSS, JavaScript

\- Backend: Python, Flask

\- База данных: SQLite
